# Горячие клавиши InstrumentTracker

## Общие клавиши

| Windows/Linux | macOS | Действие |
|--------------|-------|----------|
| **Ctrl + Q** | **⌘ + Q** | Выход из приложения |
| **Ctrl + E** | **⌘ + E** | Экспорт всех данных |
| **F5** или **Ctrl + R** | **F5** или **⌘ + R** | Обновить текущую вкладку |
| **F1** | **F1** | Открыть руководство пользователя |

## Тема оформления

| Windows/Linux | macOS | Действие |
|--------------|-------|----------|
| **Ctrl + D** | **⌘ + D** | Переключиться на тёмную тему |
| **Ctrl + L** | **⌘ + L** | Переключиться на светлую тему |

## Вкладка "Каталог активов" (только админ)

| Windows/Linux | macOS | Действие |
|--------------|-------|----------|
| **Ctrl + N** | **⌘ + N** | Добавить новый актив |
| **Ctrl + E** | **⌘ + E** | Редактировать выбранный актив |
| **Delete** | **Delete** | Удалить выбранный актив |
| **Ctrl + I** | **⌘ + I** | Импорт из Excel |
| **F5** | **F5** | Обновить список активов |
| **Двойной клик** | **Двойной клик** | Редактировать актив |

## Вкладка "Операции"

| Windows/Linux | macOS | Действие |
|--------------|-------|----------|
| **Ctrl + Shift + I** | **⌘ + Shift + I** | Выдать актив (только админ) |
| **Ctrl + Shift + R** | **⌘ + Shift + R** | Вернуть актив |
| **Ctrl + Shift + A** | **⌘ + Shift + A** | Запросить актив |
| **F5** | **F5** | Обновить историю операций |

## Примечания

- На macOS система автоматически заменяет **Ctrl** на **⌘ (Command)**
- Клавиша **F5** работает одинаково на всех платформах для обновления
- Все горячие клавиши отображаются в меню рядом с соответствующими командами
- Горячие клавиши работают, даже если соответствующие кнопки скрыты (например, для обычных пользователей)
